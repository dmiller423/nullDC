# nullDC
nullDC, a sega dreamcast emulator for windows

This was a private closed source project, never cleaned up fully for release.
The source code was released without consent by it's co-author: Stefanos Kornillios M. Poiitidis in 2005.  
Since then several derivitive works of both Reicast ('nullDC-e') and "FlyCast" (Reicast ^ retroarch) were born of it.
